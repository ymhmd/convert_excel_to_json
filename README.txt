1- For xml format, run the following line from cmd in the same directory of the python code:
    excel_script_with_comments_AND_IOS.py AND <Excel File>

2- For json format, run the following line from cmd in the same directory of the python code:
    excel_script_with_comments_AND_IOS.py IOS <Excel File>

3- The code will create "out" folder containing the generated files
